• 掌悦理财是一个什么平台？

    掌悦理财，隶属上海顽色投资管理有限公司，成立于上海陆家嘴金融中心。公司注册实缴资金8000万元，由城乡小康发展促进中心（国家事业单位）战略控股。是专业从事互联网金融财富管理的平台，为客户提供安全、透明、专业、高效的综合性投资信息中介服务。其产品丰富多样，以供应链金融、消费金融产品为主，可提供多样化的资产配置，满足投资者不同类型需求。

  • 掌悦理财如何保障投资人的本金及收益？

    掌悦理财中的定期和活期产品，精选互联网金融合作机构优质项目。与亚太保险合作，同时平台设立了风险分担机制，启动“掌盾计划”，充分保障投资人本金及收益安全。

  • 我是新手，我该如何投资？

    1.在掌悦理财官网、APP Store、应用市场下载掌悦理财APP。

    2.使用本人的手机号码进行注册。

    3.验证身份证信息，绑定与身份证信息一致的银行卡。

    5.在操作过程中如果遇到问题，可进入“常见问题“寻求帮助。如未能解决问题，可拨打客服热线400-611-8088进行咨询。

  • 什么是掌盾计划？

    掌盾计划：是指当加入“掌盾计划”的合作方发行的某期产品出现逾期时，掌悦理财将根据“掌盾计划”相关规则为相应的投资人在追讨本金及可能收益时发生的法律诉讼提供法律援助。

    除另有说明外，平台所有投资用户经平台身份认证后，在平台的投资行为均适用“掌盾计划”。

    掌悦理财有权根据实际业务需求对相关内容进行调整，如作修改，掌悦理财将及时进行披露。

  • 登录密码设置的要求？

    密码建议包含数字和英文字母（区分大小写），至少6位字符，不超过16位字符。

  • 交易密码设置的要求？

  • 登录密码和交易密码有什么区别？

    登录密码是用户登录账户时使用的密码；交易密码是用户进行申购、赎回等涉及资金操作时使用的密码。

  • 如何更换已注册的手机号码？

  • 用户账号或密码错误怎么办？

    同一账号连续输错5次密码，系统锁定该账号一天，一天后自动解锁，若已连续输错2次，建议使用密码找回功能。

  • 对获取动态密码的手机号码有什么要求？

    获取动态密码的手机号必须是您在掌悦理财平台注册的手机号码，限大陆地区电信运营商支持的号段且不支持小灵通，以及香港、台湾地区手机号码。

  • 登录失败的原因？

    请您核对以下几点，这些可能是导致您登录失败的原因：

    1.确认输入的用户名是否正确。

    2.确认输入的密码是否正确。如果忘记密码，您可以点击“忘记密码”重新设置。

    3.网络问题也会造成登录失败，如果提示网页有错误，建议您稍后尝试。

  • 姓名中有符号或者是英文是否可以认证？

    抱歉，平台暂不支持。

  • 使用护照号、军官证号是否可以认证？

    为确保客户资金安全，目前仅支持大陆居民二代身份证证件号进行认证。

  • 我认证成功后，是否可以更改证件号码？

    抱歉，平台暂不支持更改。

  • 为什么实名认证失败？

    1.核对姓名、身份证号是否对应正确。

    2.只支持中国大陆居民二代身份证，暂不支持其他认证方式。

    3.是否更改过姓名、身份证号信息等。

    4.户口是否做过迁移，户籍在落实之前会有一段冻结期。

  • 银行卡绑定失败的原因？

    1.银行卡号输入有误，请重新核对卡号输入。

    2.银行超时或银行系统异常，建议关闭网页后重新操作。

    3.输入姓名有误，可能绑定的不是实名认证一致的银行卡，建议换与实名认证一致的银行卡尝试。

    4.银行预留手机号有误，可能的原因有：

    （1）银行柜台提供的预留手机号和绑卡时输入的手机号不一致会导致鉴权失败。

    （2）银行预留的手机号只保存在用户的个人基本信息记录里面，还需要用户到柜台申请该手机号作为银行预留手机号用作快捷支付才能使用。

    （3）注意：能接收验证码的手机号不一定是柜台预留手机号，建议用户持身份证和银行卡去银行办理修改或者更换其他的银行卡绑定。

  • 申请提现，什么时候可以到账？

    当天提现，下一工作日到账，节假日申请提现最迟于T+2个工作日到账。具体到帐时间与绑卡银行处理速度有关。

  • 我可以把钱提现到别的卡上吗？

    为保证投资人的利益，投资人只能绑定与本人身份信息一致的银行卡，不能把钱提现到他人的银行卡。

  • 充值未到账该如何处理？

    1.您输入的验证码或交易密码错误：请输入正确的验证码或交易密码。

    2.银行卡余额不足：请您更换银行卡充值。

    3.充值超出限额：请联系银行查询及更改支付额度。

    4.您的银行卡未开通网上银行：请前往银行柜台或通过银行官网进行开通。

    5.您的银行卡未开通银联在线支付：请您开通银联在线支付业务后再尝试充值。

    6.页面提示超时或网络不稳定：请您切网络或者稍后再重试。

    7.验证码接收不到：请您检查手机短信屏蔽设置，或联系客服核实验证码发送情况。

    8.银行数据传输异常，导致订单超时等情况：需要刷新页面重新操作。

    9.银行卡已过期、作废、挂失：请更换银行卡或将绑定银行状态恢复正常。

    10.没有跳转到充值页面：请您检查是否浏览器拦截了弹出的页面，或者更换浏览器尝试。

    如您有遇到其他问题或以上方法未能解决问题，请联系人工客服。

    提现：代表在掌悦理财账户可用余额的资金提取至银行卡。

    赎回：代表从产品中把正在投资的资金赎回到可用余额。

  • 赎回手续费
